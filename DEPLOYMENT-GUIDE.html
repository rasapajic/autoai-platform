🚂 Deploy na Railway (korak-po-korak)

1

Napravi GitHub nalog

Idi na github.com → Sign up (besplatno).
GitHub čuva tvoj kod i Railway ga odatle preuzima.

2

Uploaduj kod na GitHub

Na GitHub.com:

  • Klikni + → New repository
  • Naziv: autoai-platform
  • Klikni uploading an existing file
  • Prevuci sve fajlove iz ZIP-ova (faza01–faza05)
  • Klikni Commit changes
3

Napravi Railway nalog

Idi na railway.app → Login with GitHub.
Ovo automatski poveže tvoj GitHub sa Railway-om.

4

Kreiraj novi projekat

Na Railway dashboardu:

  • Klikni New Project
  • Izaberi Deploy from GitHub repo
  • Izaberi autoai-platform
5

Dodaj baze podataka

U Railway projektu klikni + New i dodaj:

  • PostgreSQL — Railway automatski postavi
  • Redis — isto automatski

Railway automatski popuni DATABASE_URL i REDIS_URL env varijable! ✨

6

Postavi environment varijable

U Railway → tvoj servis → Variables tab, dodaj:

ANTHROPIC_API_KEY=sk-ant-tvoj-kljuc-ovde
SECRET_KEY=random-string-od-32-karaktera
APP_ENV=production
CORS_ORIGINS=https://tvoj-frontend.railway.app
💡 ANTHROPIC_API_KEY dobijаš na console.anthropic.com
7

Deploy!

Railway automatski builduje i deploya kada pushuješ na main granu.

Pratи buildovanje u Deployments tabu — traje 3-5 minuta.

✅ Kada vidiš ● Active — platforma je online!
8

Generiši javni URL

Railway → tvoj frontend servis → Settings → Generate Domain

Dobijaš URL tipa: autoai-platform.up.railway.app 🎉

🔧 Nakon Deploya

python -c "

scrape_portal.delay('autoscout24', {})
print('Scraping pokrenut!')
